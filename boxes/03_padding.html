Notes:

  • The previous example, margin, showed how to add spacing between blocks. This example shows how to add spacing inside blocks.

  • To add spacing inside block elements, we use the CSS property: padding.

    It has 4 options, representing each side of the box:
    (In parantheses are the 'logical properties' padding equavalents. For more information, watch this segment on Logical Properties from the Google Dev Summit )

    • padding-top ( padding-block-start )
    • padding-right ( padding-inline-end )
    • padding-bottom ( padding-block-end )
    • padding-left ( padding-inline-start )

    The following adds 10 pixels of padding to each side of the box:

    .box {
  padding-top: 10px;
  padding-right: 10px;
  padding-bottom: 10px;
  padding-left: 10px;
}

  • As with the margin example, you can employ shorthands to make the code more readible, and less verbose.

    All sides the same Top/bottom, left/right Top, left/right, bottom All sides different (T, R, B, L) 
    .box {
  padding: 10px;
}
              
    .box {
  padding: 30px 15px;
}
              
    .box {
  padding: 30px 15px 20px;
}
              
    .box {
  padding: 30px 15px 20px 10px;
}
