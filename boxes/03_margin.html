To add horizontal and vertical spacing to elements, we use the CSS property: margin.

It has 4 options, representing each side of the box:
(In parantheses are the 'logical properties' margin equavalents. For more information, watch this segment on Logical Properties from the Google Dev Summit )

  • margin-top ( margin-block-start )
  • margin-right ( margin-inline-end )
  • margin-bottom ( margin-block-end )
  • margin-left ( margin-inline-start )

The order above is notable, since that is the order you set them in the shorthand variants.

Note: Inline elements only honor left and right margins.

In the first example above (Box 1), it has 30 pixels of margin on each of its four sides. The CSS would read: 

.box {
  margin-top: 30px;
  margin-right: 30px;
  margin-bottom: 30px;
  margin-left: 30px;
}

However, this is verbose and redundant, since all sides are equal. Luckily, there are shorthands you can use to group similar values:

All sides the same Top/bottom, left/right Top, left/right, bottom All sides different (T, R, B, L) 
.box {
  margin: 30px;
}
        
.box {
  margin: 30px 15px;
}
        
.box {
  margin: 30px 15px 20px;
}
        
.box {
  margin: 30px 15px 20px 10px;
}
