09 - Border

Box borders are largely presentational, added via CSS. As you will see in the Box Model example, borders directly add to the width and height of the box. If you add a one-pixel border to the box, you have increased the box's width and height by 2 (one pixel for each opposite side).

To add borders to elements, we use the CSS property: border.

It has 4 options, representing each side of the box:
(In parantheses are the 'logical properties' margin equavalents. For more information, watch this segment on Logical Properties from the Google Dev Summit )

  • border-top ( border-block-start )
  • border-right ( border-inline-end )
  • border-bottom ( border-block-end )
  • border-left ( border-inline-start )

For example, this adds a 2 pixel, red border:

    .box {
      border-top: 2px solid red;
      border-right: 2px solid red;
      border-bottom: 2px solid red;
      border-left: 2px solid red;
    }

Or the shorthand:

    .box {
      border: 2px solid red;
    }
border: 2px solid red;

Note: 2px solid red is shorthand itself! It combines border-width, border-style, and border-color.

Width

To set the width, or thickness of the border line, use border-width.

You can set all widths individually using the longhand format:

    .box {
      border-top-width: 3px;
      border-right-width: 1px;
      border-bottom-width: 7px;
      border-left-width: 5px;
    }

Or shorthand:

    .box {
      border-width: 3px 1px 7px 5px;
    }
border-width: 2px 1px 3px 5px;

A neat trick is to set a baseline border value, then separately set the border widths after it in the cascade:

    .box {
      border: 1px solid red;
      border-width: 6px 2px;
    }
    /* For demo purposes. Don't chain classes unless absolutely necessary! */
    .box.box1 {
      border-right-width: 4px;
    }
    .box.box2 {
      border-width: 14px 1px;
    }
border-width: 6px 2px;
border-right-width: 8px;
border-width: 14px 1px;

Style

To change the style of the border line, use border-style:

    .box {
      border-style: dashed;
      border-style: dashed dotted;
      border-style: dashed dotted solid;
      border-style: dashed dotted solid double;
    }

Color

To change the color of the border line, use border-color:

    .box {
      border-color: red;
      border-color: red purple;
      border-color: red purple green;
      border-color: red purple green blue;

One interesting interaction with color is you can set individual sides to be transparent (top and bottom):

The border will still take up the space in the document. This is great if you want to add a hover effect:

Adding a border on hover when there wasn't one

.box { border:0; }
.box:hover { border:1px solid red; }
Hover over

Notice how the content jumps, since there is now a border.

Adding a border on hover from a transparent border base.

.box { border:1px solid transparent; }
.box:hover { border-color:red; }
Hover over

Now the content doesn't shift, since the border width never changes, just the color.

Arrows

A neat usage of transparent borders is to create an arrow out of CSS:

This uses some advanced CSS ideas, but the crux of it is setting all but one side's border to transparent:

    .arrow { border:8px solid transparent; border-top-color:indianred; }
    .arrow:hover { border-top-color:transparent; border-bottom-color:indianred; }
